# 🎓 Student Management System (Core Java)

A simple **console-based Student Management System** developed using **Core Java**.
This project demonstrates **Object-Oriented Programming concepts and CRUD operations**.

---

## 🚀 Features

* ➕ Add Student
* 📋 View Students
* ❌ Delete Student
* 🧾 Menu-driven console interface

---

## 🛠 Technologies Used

* Java
* ArrayList
* OOP (Classes & Objects)
* Console-based UI

---

## 📂 Project Structure

StudentManagementSystem

    └── com.sms
